# 神经表观遗传学课题组网站 - 设计规格文档

## 1. 项目概述

**项目名称**: 神经表观遗传学课题组官网
**网站类型**: 学术研究实验室官方网站
**目标用户**: 科研人员、学生、合作机构、访问学者

---

## 2. 页面结构

### 2.1 页面清单
1. **首页 (index.html)** - 实验室概览
2. **关于我们 (about.html)** - 课题组简介
3. **研究方向 (research.html)** - 研究领域详情
4. **团队成员 (team.html)** - 团队介绍
5. **联系我们 (contact.html)** - 联系信息

## 3. 设计规格

### 3.1 色彩系统

| 用途 | 颜色名称 | Hex 值 |
|------|----------|--------|
| 主色-深 | Deep Blue | #1a4a7a |
| 主色-浅 | Light Blue | #3d7ab5 |
| 辅助色 | Cyan Blue | #4a90d9 |
| 强调色 | Accent Red | #c0392b |
| 强调色-软 | Soft Red | #e74c3c |
| 背景色 | Light Gray | #f8f9fb |
| 文字色-主 | Dark | #1a1a2e |
| 文字色-次 | Secondary | #3d3d5c |
| 文字色-弱 | Muted | #6b6b8a |

### 3.2 字体系统

| 用途 | 字体 | 大小 |
|------|------|------|
| 标题 (H1) | Noto Serif SC / Source Han Serif SC | 2.2-3.6rem |
| 标题 (H2) | Noto Serif SC / Source Han Serif SC | 1.9-2.2rem |
| 标题 (H3) | Noto Serif SC / Source Han Serif SC | 1.3-1.5rem |
| 正文 | Noto Sans SC / PingFang SC | 0.9-1rem |
| 导航 | Noto Sans SC | 0.95rem |

### 3.3 间距系统

- 页面边距: 28px (移动端: 20px)
- 区块间距: 80-90px
- 卡片内边距: 2-2.5rem
- 元素间距: 0.5-1.5rem

### 3.4 组件规格

#### 卡片 (Card)
- 圆角: 16px
- 阴影: 0 4px 16px rgba(0,0,0,0.06)
- 悬浮阴影: 0 8px 32px rgba(0,0,0,0.08)
- 边框: 1px solid rgba(0,0,0,0.04)

#### 按钮 (Button)
- 圆角: 9999px (全圆角)
- 内边距: 16px 36px
- 阴影: 0 4px 20px rgba(26,74,122,0.3)
- 悬浮效果: translateY(-3px) + 阴影加深

#### 标签 (Tag)
- 圆角: 9999px
- 内边距: 6
- 背景: rgba(74px 14px,144,217,0.08)

## 4. 各页面设计规格

### 4.1 首页 (Landing Page)

#### 启动页 (Landing)
- 全屏深蓝色渐变背景 (#0a1628 → #0f2942 → #1a3d5c)
- 神经网络粒子动画效果
- Logo 居中，带浮动+发光动画
- 标题 + 副标题 + 机构名称
- 底部滚动提示箭头动画

#### 主内容区
- **Hero 区域**
  - 背景: 深蓝渐变 + 装饰圆形
  - 主标题: "探索大脑的表观遗传密码"
  - 副标题: 研究方向描述
  - CTA 按钮: "了解更多"
  - 统计数据: 60+ 论文, 15+ 项目, 20+ 合作, 10+ 奖项

- **研究方向卡片** (4列网格)
  - DNA甲基化
  - 组蛋白修饰
  - 非编码RNA
  - 染色质重塑

- **最新动态** (3列网格)
  - 日期 + 标题

### 4.2 关于我们页面

- 页面标题区: 深蓝渐变背景
- 内容区块:
  - 课题组简介
  - 研究使命 (列表)
  - 研究平台 (列表)
  - 合作交流

### 4.3 研究方向页面

- 6个研究项目卡片
- 每个卡片包含:
  - 标题
  - 详细描述
  - 标签列表

### 4.4 团队成员页面

- PI (指导教授): 2人
- 博士后: 2人
- 研究生: 4人
- 科研助理: 1人
- 成员卡片: 头像 + 姓名 + 职称 + 简介

### 4.5 联系我们页面

- 联系卡片: 地址、邮箱
- 联系表单: 姓名、邮箱、主题、留言
- 地图嵌入
- 加入我们区块

## 5. 交互规格

### 5.1 页面加载动画
- 标题: fadeInUp 0.8s
- 副标题: 延迟 0.15s
- 按钮: 延迟 0.3s
- 统计数据: 交错延迟

### 5.2 悬浮效果
- 卡片: translateY(-6~8px) + 阴影加深
- 按钮: translateY(-3px) + 光效扫过
- 导航链接: 下划线展开
- 标签: 轻微上移

### 5.3 主题切换
- 明/暗模式切换
- 存储在 localStorage

### 5.4 语言切换
- 中/英文切换
- 存储在 localStorage

---

## 6. 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| Desktop | > 900px | 正常布局 |
| Tablet | 481-900px | 2列→1列 |
| Mobile | ≤ 480px | 单列 + 移动端导航 |
